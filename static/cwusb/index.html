Firmware Update

Follow these steps to update your adapter firmware. Read all the steps.
  1. Download the new firmware file from the list below.
  2. Click the "Reboot to Update Mode" button below.
  3. Use the file explorer in your OS and open the RPI-RP2 drive.
  4. Copy the downloaded .uf2 file into the drive.
  5. Unplug and re‑plug the CW USB adapter.
